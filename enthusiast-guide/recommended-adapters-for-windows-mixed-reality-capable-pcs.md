# <a name="recommended-adapters"></a>Adaptateurs recommandés
> [!NOTE]
> Nous allons procéder à une mise à jour, en apportant des compléments à ces recommandations en matière de cartes PC Windows Mixed Reality. Vérifiez régulièrement la dernière version.

Il se peut que votre [PC compatible Windows Mixed Reality](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) ne dispose pas des ports HDMI et USB 3,0 de taille intégrale nécessaires pour connecter votre casque immersif. Si c’est le cas, vous aurez besoin d’adaptateurs pour connecter votre casque. Certains adaptateurs peuvent également nécessiter l’ajout d’un coupleur pour se connecter à votre ordinateur et/ou au câble du casque. Vous trouverez ci-dessous la liste des types d’adaptateurs dont vous pouvez avoir besoin, ainsi que des recommandations sur des modèles d’adaptateur spécifiques.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Adaptateurs Bluetooth USB compatibles Windows Mixed Reality

Si votre casque a été relâché après 2018, les adaptateurs ne sont pas nécessaires et les contrôleurs sont associés au casque.

Les appareils livrés avec 2017 ou une version antérieure ont besoin d’un adaptateur pour les contrôleurs. Si votre ordinateur n’inclut pas de Bluetooth pour associer les [contrôleurs de mouvement Windows Mixed Reality](controllers-in-wmr.md#Pair-motion-controllers), vous pouvez utiliser un adaptateur comme l’adaptateur micro-adaptateur USB Bluetooth 4,0 à faible énergie.

> [!NOTE]
> Microsoft ne fabrique, ne teste, ne certifie pas et ne garantit pas la fonctionnalité, la fiabilité ou la qualité des adaptateurs. Pour toute question, problème ou problème avec n’importe quel adaptateur, contactez le vendeur ou le fabricant de votre adaptateur.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Utilisation de Hubs USB 3,0 externes avec des casques pour Windows Mixed Reality

Les casques Windows Mixed Reality peuvent être utilisés avec un concentrateur USB 3,0 externe si votre ordinateur ne dispose pas de suffisamment de ports USB ouverts. Le concentrateur USB 3,0 externe **doit être alimenté en externe via un adaptateur CA** afin de fournir suffisamment d’énergie au casque.

Notez également les instructions suivantes pour l’utilisation d’un concentrateur USB 3,0 externe : 
* Le concentrateur USB doit être un concentrateur USB 3,0.
* Le concentrateur USB 3,0 doit être connecté à un port USB 3,0 sur votre PC.  
* Le concentrateur USB 3,0 doit être alimenté de manière externe (disposer d’un adaptateur CA fournissant une alimentation supplémentaire).
* Le concentrateur Multispeed USB 3,0 enfichable à 10 ports avec adaptateur d’alimentation 48W est connu pour fonctionner correctement avec les casques et la plupart des PC Windows Mixed Reality.
