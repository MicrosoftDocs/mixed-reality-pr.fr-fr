## <a name="experimenting-with-scale"></a><span data-ttu-id="6a76f-136">Expérimentation avec l’échelle</span><span class="sxs-lookup"><span data-stu-id="6a76f-136">Experimenting with scale</span></span>

<span data-ttu-id="6a76f-137">Les concepteurs ont expérimenté la modification de l’échelle en modifiant la taille réelle affichée de l’objet.</span><span class="sxs-lookup"><span data-stu-id="6a76f-137">Designers have experimented with modifying the scale by changing the displayed ‘real’ size of the object.</span></span> <span data-ttu-id="6a76f-138">En même temps, ils conservent une position d’objet unique pour rapprocher un objet qui se déplace vers la visionneuse sans aucun mouvement réel.</span><span class="sxs-lookup"><span data-stu-id="6a76f-138">At the same time, they maintain a single object position to approximate an object moving towards the viewer without any actual movement.</span></span> <span data-ttu-id="6a76f-139">Cela a été testé dans certains cas comme un moyen de simuler un affichage plus étroit des éléments tout en respectant les limitations de confort potentielles liées à l’affichage du contenu virtuel plus près que la « zone de confort ».</span><span class="sxs-lookup"><span data-stu-id="6a76f-139">This was tested in some cases as a way to simulate up-close viewing of items while still respecting potential comfort limitations of viewing virtual content closer than the “zone of comfort” would suggest.</span></span>

<span data-ttu-id="6a76f-140">Cela peut toutefois créer quelques artefacts possibles dans l’expérience :</span><span class="sxs-lookup"><span data-stu-id="6a76f-140">This can create a few possible artifacts in the experience however:</span></span>
* <span data-ttu-id="6a76f-141">Pour les objets virtuels qui représentent un objet avec une taille « connue » pour la visionneuse, la modification de l’échelle sans modifier la position amène aux signaux visuels conflictuels. Les yeux peuvent toujours « voir » l’objet à une profondeur en raison des indications vergence. Pour plus d’informations, consultez l’article [Comfort](comfort.md) .</span><span class="sxs-lookup"><span data-stu-id="6a76f-141">For virtual objects that represent some object with a ‘known’ size to the viewer, changing the scale without changing the position leads to conflicting visual cues. The eyes may still ‘see’ the object at some depth because of vergence cues. For more information, see the [Comfort](comfort.md) article.</span></span> <span data-ttu-id="6a76f-142">La taille agit comme une indication monoculaire que l’objet peut se rapprocher.</span><span class="sxs-lookup"><span data-stu-id="6a76f-142">The size acts as a monocular cue that the object might be getting closer.</span></span> <span data-ttu-id="6a76f-143">Ces signaux conflictuels conduisent à des perceptions confuses. les visionneuses voient souvent que l’objet reste en place (en raison de la profondeur constante) mais se multiplient rapidement.</span><span class="sxs-lookup"><span data-stu-id="6a76f-143">These conflicting cues lead to confused perceptions – viewers often see the object as staying in place (because of the constant depth cue) but growing rapidly.</span></span>
* <span data-ttu-id="6a76f-144">Dans certains cas, la modification de l’échelle est considérée comme un repère « à l’aise », alors que l’objet peut ou non être vu pour changer l’échelle par une visionneuse, mais semble bouger directement vers les yeux de la visionneuse (ce qui peut être une sensation inconfortable).</span><span class="sxs-lookup"><span data-stu-id="6a76f-144">In some cases, change of scale is seen as a ‘looming’ cue instead, where the object may or may not be seen to change scale by a viewer, but does appear to be moving directly toward the viewer’s eyes (which can be an uncomfortable sensation).</span></span>
* <span data-ttu-id="6a76f-145">Avec les surfaces de comparaison dans le monde réel, ces changements de mise à l’échelle sont parfois considérés comme la modification de la position le long de plusieurs axes. les objets semblent déplacés vers le bas au lieu de se rapprocher (comme dans une projection 2D de mouvement 3D dans certains cas).</span><span class="sxs-lookup"><span data-stu-id="6a76f-145">With comparison surfaces in the real world, such scaling changes are sometimes seen as changing position along multiple axes – objects appear to drop lower instead of moving closer (similar in a 2D projection of 3D movement in some cases).</span></span>
* <span data-ttu-id="6a76f-146">Enfin, pour les objets sans une taille « réelle » connue (par exemple, des formes arbitraires avec des tailles arbitraires, des éléments d’interface utilisateur, etc.), la modification de l’échelle peut agir de façon fonctionnelle pour imiter les modifications de distance.</span><span class="sxs-lookup"><span data-stu-id="6a76f-146">Finally, for objects without a known ‘real world’ size (for example, arbitrary shapes with arbitrary sizes, UI elements, and so on), changing scale may act functionally as a way to mimic changes in distance.</span></span> <span data-ttu-id="6a76f-147">Les visionneuses n’ont pas autant de signaux descendants préexistants pour comprendre la taille réelle de l’objet ou l’emplacement. l’échelle peut donc être traitée comme un signal plus important.</span><span class="sxs-lookup"><span data-stu-id="6a76f-147">Viewers don't have as many pre-existing top-down cues to understand the object’s true size or location, so the scale can be processed as a more important cue.</span></span>
